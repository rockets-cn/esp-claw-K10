## Supported chips

ESP-Claw requires at least 8MB Flash and 8MB PSRAM.

Currently supported:
- ESP32-S3

Coming soon:
- ESP32-P4

## Supported development boards

See [GitHub repository](https://github.com/espressif/esp-claw/tree/master/application/basic_demo/boards).
