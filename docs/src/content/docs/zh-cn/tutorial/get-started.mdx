## 准备工作

进行到这里，你应当已经组装好 ESP-Claw 面包板，或已经准备好 ESP-Claw 适配的其他硬件。
你可以在[GitHub 仓库](https://github.com/espressif/esp-claw/tree/master/application/basic_demo/boards)中查看 ESP-Claw 适配的开发版列表。
ESP-Claw 需要至少 8 MB Flash 和 8 MB PSRAM，目前支持 ESP32-S3，正在适配 ESP32-P4。

在烧录之前，你还需要准备一台电脑，并安装 Chrome 或 Edge 浏览器。在线烧录暂时无法使用 Firefox 浏览器完成。

<Aside type="tip">
    **进阶操作**: [从源码编译与烧录](@lang/reference-project/build-from-source)。
</Aside>

## 在线烧录

<Steps>

1. **使用数据线连接开发版**

   使用数据线将开发版连接到 PC。如果你使用面包板形态的 ESP-Claw，请使用标记有 UART 的 USB 口连接。

2. **启动在线烧录工具并连接串口**

   <LinkCard title="在线烧录" href="@lang/flash" description="打开在线烧录工具，为 ESP32 烧录 ESP-Claw" />

   打开在线烧录工具，点击「连接」，连接到开发版。

3. **配置参数**

   - 选择你的开发版对应的固件。
   - 配置 Wi-Fi SSID 及密码，必填。
   - 配置 LLM：可选 GPT (OpenAI), Qwen (阿里云百炼), Claude (Anthropic)，请按提示生成 API Key 并填入。
      <Aside type="note" title="请选择推理能力较强的模型">
         ESP-Claw 的自编程、复杂工具组合等功能依赖强推理模型的能力。
         推荐选用 GPT-5.4、Qwen3.6-plus、Claude3.5-sonnet 或类似性能的模型以取得最佳体验。
      </Aside>
   - 配置需要连接到的聊天软件：目前支持 Telegram、QQ Bot（OpenClaw）、飞书与微信 ClawBot 等，可同时填写多个。
     | 聊天软件 | 配置项 | 帮助文档 |
     | --- | --- | --- |
     | Telegram | Bot Token | [Telegram Bot 文档](https://core.telegram.org/bots#6-create-a-token) |
     | QQ Bot | ID 和 Secret | [QQ Bot 网站](https://q.qq.com/qqbot/openclaw/) |
     | 飞书 | App ID 和 Secret | [飞书文档](https://www.feishu.cn/content/article/7602519239445974205) |
     | 微信 ClawBot | 微信扫码自动配置 | N/A |
   - 配置搜索引擎:
      <Aside type="tip">
         配置搜索引擎后，ESP-Claw 才能通过网络检索获取最新信息。查询天气也依赖搜索引擎支持。

         推荐配置 Tavily，有一定的免费额度。
      </Aside>
      打开 [Tavily Dashboard](https://app.tavily.com/)，注册账号并登录，复制一个 API Key 并填入网页烧录工具的「Tavily」输入框。
   - 配置高级设置（可选），见 [配置说明](@lang/reference-project/configuration)。

4. **烧录**

   点击「烧录」按钮，开始烧录。

5. **开始体验！**

   烧录完成后，设备会自动重启，即可开始体验。

   <Aside type="note">
      如输入了错误的 Wi-Fi SSID 或密码，ESP-Claw 可能无法联网。此时，可以连接 ESP-Claw 提供的 SoftAP 网络，检查并修改配置。[➡️Web 配置](@lang/tutorial/web-config)
   </Aside>

</Steps>

## 上手体验

<Aside type="tip">
   带屏设备可以通过屏幕状态判断是否已连接到 Wi-Fi。无屏设备需要通过串口 Console 或 Web 配置页面的装显示判断是否已连接到 Wi-Fi。
</Aside>

推荐使用 Telegram / QQ Bot / 微信 ClawBot 与 ESP-Claw 交互。
- 使用已配置的 IM 通道发送消息，ESP-Claw 会自动回复。
- 无法获得回复？[➡️FAQ](@lang/tutorial/faq#通过-im-发送的消息没有响应)
- 希望通过串口 Console 交互？[➡️Console 交互](@lang/reference-project/console-usage)

**日常问答**
- 你好，介绍一下你当前有哪些能力？
- 最近有什么 AI 新闻？
- 帮我计算 12% 年化利率，3年后10000元会变成多少钱？

**控制硬件**（需硬件支持）
- 打开灯 / 把灯调成红色
- 在屏幕上写 `Hello, ESP-Claw!`
- 拍一张照片发给我
- 播放 1s 440Hz 的提示音
- 在屏幕上画一个笑脸

**定时任务**
- 每天早上 7 点提醒我喝水
- 每天晚上 9 点提醒我睡觉
- 每周六早上 8 点提醒我打扫卫生

**询问记忆**
- 记住我喜欢简洁的回答
- 你还记得我叫什么名字吗？
